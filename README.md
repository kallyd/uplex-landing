# UplexOS Landing Page

Landing page institucional independente do kernel UplexOS.

## Desenvolvimento

```powershell
npm install
npm run dev
```

Acesse `http://localhost:3000`.

## Verificação

```powershell
npm test
npm run typecheck
npm run build
```

A instalação de dependências requer acesso ao registry npm. O projeto não usa imagens remotas, analytics ou APIs externas.
